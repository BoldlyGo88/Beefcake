# ModMaterialsFileAdd

```lua
ModMaterialsFileAdd("date/stockmodfolder/morematerials.xml");
```

# Description

Appends more materials to the standard materials already loaded by the game

# Parameters

- filename: file path to the extra materials ***[string]***

# Return Value

N/A
